# NodeJSSimpleTypeScriptBoilerPlate
Get started using TypeScript with Node.JS with no config.

##Getting Started


### Install

      npm i -g typescript

### Run
If you are using node.js run

    npm i
    
If you are using yarn run 

    yarn
    
    
###
Thats it all there is left to do is to run 

    node start
